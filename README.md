## 🧠 About Me

- 🔭 Researching **Urdu Document Intelligence** — a VLM (Qwen3) + fine-tuned ByT5 pipeline for extracting and correcting Urdu text from scanned documents
- 🧩 Building **Specter**, an AI-powered RAG-based legal assistant for Pakistani lawyers (Final Year Project, MPVIR Group)
- 🤖 Designing **Multi-LLM RAG systems** with user-selectable models, embeddings, and vector databases
- 🌐 Full-stack development across MERN and modern web stacks
- 🎓 Active in IEEE RAS, GDGoC, GitHub Campus Experts, and Amnesty International at COMSATS
- 💬 Focused on **NLP for low-resource languages** (especially Urdu) and legal tech
